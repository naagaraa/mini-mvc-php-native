Gagal Terhubung ke Server

check file .env pada folder root atau project ada atau tidak ?
jika tidak ada silahkan buat file .env atau bisa copy dari file .env.example
atau bisa juga menggunakan command nagara pada terminal atau command promt atau copy saja perintah berikut ini

Aturan atau how to fixed :

C:\xampp\htdocs\projectdir> php nagara generate:copyenv

C:\xampp\htdocs\projectdir> php nagara generate:env

Copy perintah berikut ini dan paste pada terminal atau powershell :
  1. php nagara generate:copyenv
  2. php nagara generate:env
